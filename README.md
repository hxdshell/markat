# Markat - Email Client

## Fundamental Features

- [ ] Navigate mailboxes
- [ ] Fetch Envelopes and paginate
- [ ] Render the message body and attachments
- [ ] Manage metadata of a message
- [ ] Manage threads
- [ ] Implement Search
- [ ] SMTP for sending mails
- [ ] Secure Authentication

## More ambitious features

- [ ] Manage multiple accounts at once
- [ ] Custom themes

## Quality attributes

- [ ] User Interface should be snappy.
- [ ] Test modules and display errors where needed. Errors should not be ignored.
